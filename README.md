# Oscars Picks

Oscars Picks web app to allow for picks parties.

Create a room, invite friends, make your picks, and watch the leaderboard update live as winners are announced.

## Setup

1. Copy `.env.example` to `.env` and fill in your Firebase config values
2. `npm install`
3. `npm run dev`

## Deploy

Build command: `npm run build`
Publish directory: `dist`
